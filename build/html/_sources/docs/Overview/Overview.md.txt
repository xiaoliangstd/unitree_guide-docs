## Overview 
The unitree_guide project is the open source software framework accompanying [《四足机器人控制算法--建模、控制与实践》](https://detail.tmall.com/item.htm?id=704510718152) published by Unitree Robotics.This document will give you some details and help you get started with unitree_guide.
